Projects

Pulse Oximeter

Developed a real-time pulse oximeter using Arduino Uno R3 to monitor heart rate (BPM) and blood oxygen saturation (SpO₂).

Door Security Alarm System

Designed and implemented an IR sensor–based door security alarm system to detect unauthorized entry and trigger alerts.

Automated Plant Watering System

Developed an automated plant watering system using an Arduino and a moisture sensor. A 5V single-channel relay is used to control a 9V DC motor for watering plants based on soil moisture levels.

Components Used: Arduino, Moisture Sensor, 5V 1-Channel Relay Module, 9V DC Motor

IoT-Based Health Monitoring System

Developed an IoT-based health monitoring system using an ESP32 to measure heart rate and blood oxygen saturation levels. The MAX30102 sensor is used for data acquisition, and real-time values are displayed on a 0.96-inch OLED display. The health data is uploaded to ThingSpeak for remote monitoring.

Components Used: ESP32, MAX30102 Sensor, 0.96" OLED Display, ThingSpeak Cloud Platform

Body Posture Detection System

Developed a body posture detection system using a flex sensor and ESP32 to identify good and bad posture. The system activates a buzzer when incorrect posture is detected, and posture status is displayed through serial output in the Arduino IDE.

Components Used: ESP32, Flex Sensor, Buzzer

Ultrasonic Distance Alert System

Developed a distance alert system using an ultrasonic sensor and Arduino Uno R3 to measure the distance of nearby objects. A buzzer provides an alert when an object comes within a predefined range, and the measured distance is displayed on an I2C LCD.

Components Used: Arduino Uno R3, Ultrasonic Sensor, Buzzer, I2C LCD Display
